# Identity Verification

## Issue
Customer requests account-specific assistance and identity verification is required before account changes or detailed support actions.

## Symptoms
- Customer needs account access help
- Customer requests changes to contact details
- Customer asks about a service linked to a personal account
- Sensitive information may need to be verified before proceeding

## Diagnosis
1. Confirm that the request concerns an account-specific action.
2. Do not ask the customer to post private information publicly.
3. Move the exchange to a private channel if verification is needed.
4. Follow the required verification process before any account change is made.

## Recommended Agent Action
- Ask the customer to continue through a secure or private support channel.
- Verify identity before discussing sensitive details or changing account information.
- If verification cannot be completed, explain that the request cannot proceed yet.

## Escalation Criteria
- Customer cannot complete verification
- Account recovery requires specialist support
- Sensitive account change request remains unresolved

## Suggested Reply
For account-specific assistance, I’ll need to continue through a secure private channel so we can complete verification safely. Once your identity is confirmed, we can proceed with the requested account update.
